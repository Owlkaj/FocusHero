Nama dan Deskripsi Aplikasi
Nama Aplikasi: FocusHero
Deskripsi Aplikasi:
FocusHero merupakan aplikasi web berbasis Gamified Habit Tracker yang dirancang untuk membantu pengguna dalam membangun dan mempertahankan kebiasaan positif sehari-hari, seperti belajar, membaca, dan berolahraga. Aplikasi ini menggabungkan konsep manajemen kebiasaan dengan elemen gamifikasi untuk meningkatkan motivasi dan konsistensi pengguna dalam mencapai tujuan mereka.
Tema Aplikasi:
Tema yang diangkat dalam aplikasi ini adalah produktivitas dan pengembangan diri (self-improvement), dengan pendekatan gamifikasi yang memberikan pengalaman interaktif dan menarik bagi pengguna.

Daftar Fitur Utama
Sistem autentikasi pengguna yang meliputi registrasi, login, dan logout.
Manajemen kebiasaan pengguna, seperti menambahkan, melihat, dan mengelola kebiasaan harian.
Fitur penandaan kebiasaan sebagai selesai untuk mencatat aktivitas yang telah dilakukan.
Sistem gamifikasi berupa pemberian poin berdasarkan kebiasaan yang berhasil diselesaikan.
Dashboard pengguna yang menampilkan ringkasan aktivitas dan perkembangan kebiasaan.
Pengelolaan data pengguna oleh admin (jika tersedia dalam sistem).

Teknologi yang Digunakan
Bahasa pemrograman JavaScript dengan lingkungan runtime Node.js.
Framework Express.js untuk membangun backend aplikasi.
Database MySQL untuk penyimpanan data.
HTML dan Tailwind CSS untuk tampilan antarmuka pengguna.
dotenv untuk pengelolaan konfigurasi environment.
phpMyAdmin sebagai alat bantu dalam manajemen database.

Cara Penggunaan Aplikasi
Aplikasi diakses melalui browser pada alamat focushero-production.up.railway.app.
Pengguna dapat melakukan registrasi untuk membuat akun baru atau login jika sudah memiliki akun.
Setelah login, sistem akan menampilkan halaman utama sesuai dengan peran pengguna, yaitu sebagai admin atau anggota.

Sebagai Admin
Admin memiliki hak akses penuh terhadap sistem.
Admin dapat melihat dashboard yang berisi ringkasan aktivitas dan statistik kebiasaan pengguna.
Admin dapat mengelola data pengguna, seperti melihat daftar pengguna yang terdaftar.
Pada implementasi tertentu, admin dapat mengatur atau menambahkan jenis kebiasaan yang digunakan oleh seluruh pengguna.

Sebagai Anggota (User)
Pengguna dapat menambahkan kebiasaan baru sesuai kebutuhan, seperti belajar, membaca, atau olahraga.
Kebiasaan yang telah dibuat akan ditampilkan pada dashboard pribadi pengguna.
Pengguna dapat menandai kebiasaan sebagai selesai setelah dilakukan.
Setiap kebiasaan yang diselesaikan akan menambah progress dan poin sebagai bagian dari sistem gamifikasi.
Pengguna dapat memantau perkembangan kebiasaan melalui dashboard yang tersedia.
